# List of Player Skills

This section lists all {term}`Skill`s that are available to 
{term}`Player Character`s.


(skills:list:core)=
## The Core Skills

All {term}`Character`s have access to all {term}`Core Skill`s and automatically
have a {term}`Level` of of 1 in each.

* [Fitness](skills:core:fitness)
  * [Strength](skills:core:strength)
  * [Endurance](skills:core:endurance)
  * [Agility](skills:core:agility)
* [Harmony](skills:core:harmony)
  * [Communication](skills:core:communication)
  * [Deception](skills:core:deception)
  * [Empathy](skills:core:empathy)
* [Acumen](skills:core:acumen)
  * [Instinct](skills:core:instinct)
  * [Observation](skills:core:observation)
  * [Memory](skills:core:memory)

(skills:list:path)=
## The Path Skills

All {term}`PC`s and some {term}`NPC`s have access to the {term}`Path Skill`s.
{term}`Path Skill`s represent means of accessing the 
{term}`Domain Skill`s, which are typically each compatible with a subset
of {term}`Path Skill`s.

* [Intellect](skills:path:intellect)
  * [Alchemy](skills:path:alchemy)
  * [Artifice](skills:path:artifice)
* [Intention](skills:path:intention)
  * [Will](skills:path:will)
  * [Channeling](skills:path:channeling)
* [Intuition](skills:path:intuition)
  * [Attunement](skills:path:attunement)
  * [Virtue](skills:path:virtue)
* [Invocation](skills:path:invocation)
  * [Chanting](skills:path:chanting)
  * [Rituals](skills:path:rituals)
* [Intimation](skills:path:intimation)
  * [Performance](skills:path:performance)
  * [Artistry](skills:path:artistry)
